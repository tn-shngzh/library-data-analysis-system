# 🚀 快速启动指南

## ⚡ 最简单的方式

### 方式 1：使用启动脚本（推荐新手）

双击运行：
```
start.bat
```

然后按照菜单提示操作即可！

### 方式 2：命令行直接运行

```bash
# 获取问题并标注
dpo-gen fetch -p "您的问题"
```

## 📋 完整示例

### 示例 1：获取一个问题并标注

```bash
dpo-gen fetch -p "如何学习 Python？"
```

**执行流程：**
1. ✅ 模型生成问题和回答
2. 📺 显示问题和模型回答
3. ✍️ **您输入偏好的回答**
4. 💾 自动保存为 JSON

**输出 JSON：**
```json
{
  "prompt": "如何学习 Python？",
  "chosen": "您的回答内容",
  "rejected": "模型生成的回答"
}
```

### 示例 2：连续获取多个样本

```bash
dpo-gen fetch -p "问题 1"
dpo-gen fetch -p "问题 2"
dpo-gen fetch -p "问题 3"
```

### 示例 3：查看和导出

```bash
# 查看统计
dpo-gen stats

# 导出数据
dpo-gen export -o my_data.jsonl
```

## 📁 数据保存位置

- **标注数据**: `data/annotated/dpo_data.jsonl`
- **导出数据**: `data/exports/` 或您指定的路径

## ⌨️ 标注界面操作

在标注界面中：

| 操作 | 命令 |
|------|------|
| 填写回答 | 直接输入文字 |
| 跳过 | 输入 `:q` 或 `:skip` |
| 退出 | 输入 `:quit` 或 `:exit` |
| 查看统计 | 输入 `:stat` |

## 🔧 配置（可选）

默认使用模拟模式，无需配置。

如需使用真实 API：

```bash
dpo-gen init
```

然后按照提示设置：
- API 地址
- API 密钥
- 模型名称

## 💡 使用技巧

1. **批量收集**: 准备一个问题列表，逐个运行 `dpo-gen fetch -p "问题"`
2. **质量保证**: 认真填写每个回答，确保数据质量
3. **定期导出**: 使用 `dpo-gen export` 定期备份数据
4. **查看统计**: 使用 `dpo-gen stats` 了解数据收集进度

## ❓ 常见问题

**Q: 我需要配置什么才能开始吗？**  
A: 不需要！默认使用模拟模式，可以直接测试。

**Q: 如何查看生成的数据？**  
A: 打开 `data/annotated/dpo_data.jsonl` 文件查看。

**Q: 数据格式是什么？**  
A: JSON Lines 格式，每行一个 JSON 对象。

**Q: 如何导出数据用于训练？**  
A: 运行 `dpo-gen export -o output.jsonl`
