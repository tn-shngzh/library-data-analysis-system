# 🤖 DPO 数据生成器 - 自动生成问题版

## ✅ 完成！

现在程序已经按照您的需求实现了：

### 📋 完整工作流程

```
1. [AI] 模型自动生成一个问题
   ↓
2. [AI] 模型生成一个普通回答
   ↓
3. [您] 您填写更好的回答
   ↓
4. [保存] 自动保存为 JSON 格式
```

### 🚀 启动方式

```bash
cd d:\library-data-analysis-system\dpo-data-generator
python run.py
```

### 📝 使用示例

运行后，您会看到菜单：

```
请选择操作:
1. [生成] 生成问题并标注
2. [统计] 查看统计
3. [导出] 导出数据
4. [退出] 退出程序

输入选项 (1-4):
```

**选择 1 后的流程：**

```
正在生成问题...

[问题] 生成的问题:
   如何保持学习动力？

正在生成模型回答...

[模型回答]:
   保持学习动力需要设定明确的目标，制定合理的学习计划，并定期回顾进展。

============================================================
[您的回答]
请输入您认为更好的回答（直接输入，回车结束）:
> 保持学习动力的关键是找到学习的意义和乐趣。首先，设定具体可行的目标，
  将大目标分解为小步骤，每完成一步就给自己一些奖励。其次，寻找学习伙伴
  或加入学习社群，相互鼓励和监督。最后，保持好奇心，将学习与实际生活
  联系起来，让知识变得生动有趣。

============================================================
[成功] 保存成功！
[文件] data\annotated\dpo_data.jsonl

生成的 JSON:
{
  "prompt": "如何保持学习动力？",
  "chosen": "保持学习动力的关键是找到学习的意义和乐趣...",
  "rejected": "保持学习动力需要设定明确的目标..."
}
============================================================
```

### 💾 生成的 JSON 格式

```json
{
  "prompt": "模型生成的问题",
  "chosen": "您填写的优质回答",
  "rejected": "模型生成的普通回答"
}
```

### 📊 数据统计

选择选项 2 可以查看：

```
[数据统计]
============================================================
  已标注样本数：10
  状态分布:
    [已标注] annotated: 10
  数据文件：data\annotated\dpo_data.jsonl
============================================================
```

### 💾 导出数据

选择选项 3 可以导出数据用于训练：

```
请输入输出文件名 (默认：dpo_data.jsonl):
> my_dpo_data.jsonl

[成功] 导出了 10 个样本到：my_dpo_data.jsonl
```

### 🎯 快捷操作

在输入回答时：
- 输入 `:q` 或 `:skip` - 跳过当前问题
- 输入 `:quit` 或 `:exit` - 退出程序

### 📁 重要文件

- **启动脚本**: [`run.py`](file:///d:/library-data-analysis-system/dpo-data-generator/run.py)
- **数据文件**: `data/annotated/dpo_data.jsonl`
- **导出文件**: `data/exports/` 或您指定的路径

### 🎉 立即开始

```bash
cd d:\library-data-analysis-system\dpo-data-generator
python run.py
```

然后选择 `1` 开始生成第一个样本！

### 🔧 配置真实 API（可选）

如需使用真实模型（如 OpenAI 或本地模型），运行：

```bash
dpo-gen init
```

按照提示配置 API 地址和密钥。
