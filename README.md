# 图书馆数据分析系统

## 项目结构

- `library_data_analysis_fastapi/` - FastAPI 后端
- `library_data_analysis_vue/` - Vue 3 前端

## 启动方式

### 后端
```bash
cd library_data_analysis_fastapi
pip install -r requirements.txt
uvicorn main:app --reload
```

### 前端
```bash
cd library_data_analysis_vue
npm install
npm run dev
```
